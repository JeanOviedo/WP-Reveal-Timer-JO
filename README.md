## ğŸ“– How to Use

Simply wrap the content you want to hide with the `[reveal_timer]` shortcode.

### Basic Example:
```html
[reveal_timer seconds="10"]
    <button>Get the Offer Now!</button>
[/reveal_timer]
```

### Custom Label Example:
```html
[reveal_timer seconds="30" label="Exclusive content in:"]
    <p>Success! Here is your secret download link: [link]</p>
[/reveal_timer]
```
