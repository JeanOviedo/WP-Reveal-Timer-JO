Reveals hidden content after a countdown. Ideal for flash offers, sales funnels, and timed premium content.

== Description ==

Reveal Timer JO allows you to insert a timer via shortcode. When the time ends, the counter disappears and the hidden content is automatically revealed.

* No external libraries (Vanilla JS).
* Compatible with Elementor and Gutenberg.
* Supports multiple instances on the same page.
* Modern and clean design.
* Fully customizable (Colors, Sizes, Border Radius).
* Translation-ready.

== Installation ==

1. Upload the `reveal-timer-jo` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the 'Reveal Timer' menu in the WordPress sidebar for usage instructions.

== Shortcode ==

`[reveal_timer seconds="10" label="Content available in:"] CONTENT [/reveal_timer]`

== Examples ==

* **Flash Sales:** Reveal a purchase button after a specific number of seconds to increase conversions.
* **VSL (Video Sales Letters):** Synchronize the appearance of CTA buttons with your video content.
* **Timed Access:** Show links or restricted information after a waiting period.
